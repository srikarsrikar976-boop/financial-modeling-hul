
# financial-modeling-hul
# 📊 Financial Modeling & Valuation – Hindustan Unilever Ltd

This project presents a complete **Financial Modeling and Valuation analysis** of Hindustan Unilever Ltd (HUL), covering financial statements, forecasting, valuation techniques, and risk analysis.

---

## 🚀 Project Overview

The objective of this project is to evaluate the intrinsic value of HUL using:

- Financial statement analysis  
- Ratio analysis  
- Forecasting techniques  
- Discounted Cash Flow (DCF)  
- Relative valuation  

---

## 🏢 Company Overview

Hindustan Unilever Ltd (HUL) is one of India's largest FMCG companies, operating across:

- Personal care  
- Home care  
- Food & beverages  

---

## 📌 Key Analysis Covered

### 📊 Financial Analysis
- Income Statement, Balance Sheet, Cash Flow
- Common-size statements
- Margin analysis (EBIT, EBITDA, Net Profit)

### 📈 Ratio Analysis
- ROE, ROCE, Profitability ratios  
- Turnover ratios  
- Liquidity & efficiency metrics  

### 🔮 Forecasting
- Revenue growth projection  
- EBITDA & EPS forecasting  
- Scenario-based assumptions  

### 💰 Valuation Techniques

#### 1. Discounted Cash Flow (DCF)
- FCFF model used  
- WACC calculated using CAPM  
- Terminal value with growth rate  

👉 Intrinsic Value per Share ≈ **₹2222**  
👉 Current Market Price ≈ **₹2529**  

➡️ Conclusion: Stock appears **slightly overvalued**

---

#### 2. Relative Valuation
- Compared with peers:
  - ITC  
  - Nestle India  
  - Britannia  
  - Dabur  
  - Marico  

- Metrics used:
  - EV/EBITDA

  - EV/Sales
